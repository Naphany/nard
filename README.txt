Проект - игра нарды.

Целью проекта являетя 
 • развить стратегическое мышление и навыки общения;
 • закрепить навыки счёта и ориентировки в пространстве;
 • снизить тревожность;
 • повысить самооценку;
 • стабилизировать эмоциональную сферу.

Правила игры 
 1) Игрок, набравший большее число при втором броске, делает ход, используя только числа, указанные на игральных костях, использованных для второго броска.
 2) Игроки делают поочередные ходы, бросая по два кубика в начале каждого хода.
 3) После броска кубика игроки должны, по возможности, переместить свои фигуры в соответствии с номером, указанным на каждом кубике.
 4) Если игрок выбрасывает два кубика с одинаковым числом, которые называются удвоениями, этот игрок должен разыграть каждый кубик дважды.
 5) При любом броске игрок должен двигаться в соответствии с числами на обеих костях, если это вообще возможно.
 6) Если одно или оба числа не позволяют сделать разрешенный ход, игрок проигрывает эту часть броска, и ход заканчивается.
 7) Если ходы могут быть сделаны либо одним кубиком, либо другим, но не обоими сразу, необходимо использовать большее число.
 8) Если один кубик не может быть перемещен, но такой ход становится возможным благодаря перемещению другого кубика, этот ход является обязательным.

Используемые библиотеки.

1) Pygame — набор модуля языка, предназначенный для написания компьютерных игр и мультимедиа-приложений. Pygame базируется на мультимедийной библиотеке SDL.
  - import pygame
2) Random -  содержит множество функций, связанных с эмуляцией случайности (например, "перемешивание" элементов последовательности), а не только функции генерации псевдослучайных чисел.
  - import random

Основные модули/функции, используемых библиотек:

1) main -- используется для выполнения пакетов.
2) chek_key -- правила хождения пешек.
3) winning_game --задаёт параменты/условия, при которых игрок выигрывает.
4) game -- условия игры.
